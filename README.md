Music Player Interface – Project Overview

The Music Player Interface is a web-based application developed using HTML, CSS, and JavaScript that allows users to play audio songs through an attractive and interactive interface. The project includes features such as play, pause, next, previous, and automatic song switching with album cover display.

Objective

The main objective of this project is to create a simple and responsive music player that demonstrates audio handling and multimedia functionality using JavaScript.

Features
Play and pause music
Next and previous song controls
Automatic next song playback
Album cover image display
Song title and artist name display
Responsive and modern user interface
Audio control support
Technologies Used
HTML – Structure of the music player
CSS – Styling and responsive design
JavaScript – Audio functionality and controls
How the Project Works
The application loads a list of songs stored in JavaScript.
When the user clicks the Play button, the selected audio file starts playing.
The Next and Previous buttons switch between songs.
Song details and album images update dynamically.
When a song ends, the next song automatically starts playing.
